
:or-description: A OR (Office Responsible) is an ASAM employee focussed on coordinating project processes and the background work behind a project.

:project-member-description: An xref:role-specific:standard_user.adoc[ASAM User] who is involved directly in the development of an ASAM standard. \
Every project needs project members from at least three (3) different companies. \
See also xref:role-specific:project-participant.adoc[Subgroup Member]

:project-lead-description: The person responsible for leading and coordinating all project activities. \
Without an elected Project Lead, the project may not start. \
The project members must elect a Project Lead at the beginning of the project.

:project-participant-description: A project member working in a particular sub-group, a subdivision of the overall project. \
If the project is split up into smaller work groups, a project member can join a sub-group to become a sub-group member.

:sg-lead-description: If the project is split up into smaller work groups, a project member can be elected to be the sub-group lead and will support the xref:role-specific:project-lead.adoc[Project Lead]. \
Each sub-group should have zero or one sub-group lead.

:service-provider-description: Provides additional services necessary for the standard development. \
If the project needs additional resources, a Service Provider can be hired, which requires an approved budget by the xref:role-specific:tsc.adoc[TSC].

:technical-writer-description: Formats and writes the document(s) of the standard based on the provided input from the xref:role-specific:project-participant.adoc[WG members] in a formalized and standard-conform way. \
If the project requires a technical writer, either the ASAM technical writer will fill this role or a xref:role-specific:service-provider.adoc[Service Provider] can be hired to fill that role.

:standard-user-description: An ASAM Standard user is someone who is part of the ASAM community but not involved in the project development at hand.

:standard-manager-description: The manager at ASAM responsible for the standard.

:tsc-description: The TSC is responsible for approving or rejecting budgets, projects and proposals as well as releases of project results, such as standards or concept papers.

:reviewer-description: An xref:role-specific:standard_user.adoc[ASAM Standard user] or xref:role-specific:project-participant.adoc[WG member] who reviews a draft or release candidate for a new or updated standard.
