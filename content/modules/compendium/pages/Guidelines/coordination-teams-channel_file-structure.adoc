[source]
----
├───General // this is where all administrative topics are stored
│   ├───Project_report.docx // the latest project report
│   └───Participant_list.xlsx // contains all enrolled project members with their contact details and their subgroup assignments
│
├───01_SubgroupName1 // each subgroup creates and manages their own directory as they see fit
├───02_SubgroupName2
...
└───xx_SubgroupName<N>

----

Each Sharepoint may also contain an archive containing information from previous project iterations.
