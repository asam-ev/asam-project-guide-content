= Default Trunk Name
:description: Defines the name for the default branch, the trunk.
:keywords: definition, trunk

The default branch, also known as the "trunk", is used as the central branch that represents the official status or release of a repository.
Depending on the repository, there may be additional branches for development or specific releases/versions.
They do not use this default name.
