== Result
* The Kick-off Workshop is prepared at least 2 weeks ahead of time.
* For an onsite workshop, a sufficient meeting room has been booked at the ASAM office (max. duration: 8h).
* (OPTIONAL) For an online workshop, an online event has been set up (max. duration: 4h).
+
NOTE: ASAM strongly recommends an onsite workshop for the Kick-off.
+
* The agenda covers the following targets:
** Project working model
** Elect Project Lead
** Define Work Groups
** Elect Work Group Leads
** Assign project members to work groups


== How
. Use the template to set up the agenda
. Add or update items where necessary
. Send the agenda to all enrolled people, using the enrollment list or, if already available, the MS Teams mailing list
