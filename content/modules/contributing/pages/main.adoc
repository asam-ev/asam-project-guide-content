All the tools that are required for participating in ASAM projects are hosted online and can be accessed through any web browser. +
The only thing you need to do is xref:account_setup.adoc[set up your repository account] and you can start right away - no installation required!
Just make sure to familiarize yourself with the xref:contribution_workflow.adoc[ASAM Project Guidelines] before you start contributing.

If you prefer, you may also choose to work with compatible tools on your local computer.
Advanced users can check out the xref:compendium:Tools.adoc[] section in the Compendium for further recommendations.
